## How to Use These Templates

### Before Each Session
1. Read the template for the session
2. Customize questions based on pre-discovery information
3. Share pre-read materials with participants
4. Confirm attendees and logistics

### During Each Session
1. Follow the agenda structure
2. Use the question guide as a framework
3. Document answers in the template
4. Capture parking lot items
5. Note follow-up actions

### After Each Session
1. Complete the template within 24 hours
2. Share notes with participants for validation
3. Follow up on action items
4. Update discovery documentation

## Template Components

Each template includes:
- **Session Overview**: Objectives, duration, participants
- **Pre-Session Preparation**: Materials, access, pre-reads
- **Agenda**: Time-boxed activities
- **Question Guide**: Structured questions to ask
- **Capture Sections**: Areas to document findings
- **Worksheets**: Forms and tables to complete
- **Follow-Up**: Action items and next steps
